1. HTML:

  1. O que é HTML?

É uma linguagem de marcação baseada em hipertexto - textos com hiperlinks ou links que levam de um conteúdo para outro;

Com HTML, podemos escrever documentos que serão acessados pelo navegador, portanto, é a linguagem da web;

O HTML é o esqueleto da página.

  1. O que são as TAGS?

São elementos que identificam e organizam um conteúdo dentro de um arquivo HTML através de uma marcação entre os sinais de < e >;

Elas que informam ao navegador que tipo de estrutura está sendo construída, podendo conter títulos, parágrafos, imagens, entre outros;

Um elemento HTML é uma TAG.

Geralmente vêm em pares, com uma tag de abertura (< tag>) e uma tag de fechamento (< /tag>), com o conteúdo da tag entre elas;

Alguns elementos podem ter atributos, que fornecem informações adicionais sobre como o elemento deve ser exibido ou comportar-se.

Os elementos HTML podem ser classificados em dois tipos, de acordo com o seu posicionamento em tela: elementos de bloco e elementos de linha.

  • Elementos de Bloco (block): são os elementos que ocupam todo o espaço horizontal disponível no lugar que estão inseridos. Eles geralmente se iniciam em uma nova linha e empurram todos os elementos subsequentes para baixo.
    Exemplos: h1 ao h6, p, div, section, footer
  • Elementos de Linha (inline): são os elementos que ocupam apenas o espaço necessário para exibir o seu conteúdo. Eles geralmente são exibidos um ao lado do outro e não iniciam uma nova linha.
    Exemplos: a, img, button, span, strong, input

Para descobrir qual o tipo da tag (inline ou block), basta colocar um texto ao lado da tag de fechamento e ver como a tag "reage", por exemplo:

  • < h1> Texto < /h1> Texto 2

  1. O que é a Semântica?

A Semântica refere-se à utilização de elementos que têm significado específico e claro para descrever o conteúdo da página da web de uma maneira estruturada;

Ao utilizar a semântica correta em HTML, você está indicando claramente qual parte do seu conteúdo é um cabeçalho, um parágrafo, uma lista, uma seção, uma citação, etc.

Os benefícios de utilizar a Semântica são:

  • Ajuda os navegadores a renderizarem corretamente o conteúdo;
  • Melhora a acessibilidade para usuários com deficiência;
  • Facilita a indexação por parte de mecanismos de busca e promove uma melhor compreensão do conteúdo por parte de desenvolvedores.

2. CSS:

  1. O que é CSS?

A sigla CSS significa Cascading Style Sheets;

É uma linguagem de estilos em Cascata que descreve como os elementos HTML devem ser exibidos em tela;

Através do CSS é possível definir propriedades como cor, fonte, tamanho, espaçamento e muitos outros aspectos visuais da página;

O CSS pode ser descrito como a Roupa ou Maquiagem da página, é o que traz o Estilo.

A sintaxe básica do CSS consiste em regras que definem como os elementos HTML devem ser apresentados no navegador:

  1. Seletor: É o elemento HTML ao qual você deseja aplicar estilos;
  2. Bloco de Regras: Um conjunto de declarações dentro de chaves ({ });
  3. Propriedade: A característica que você deseja alterar no elemento HTML;
  4. Valor: O valor que você deseja atribuir à propriedade;
  5. Declaração: É a combinação de uma propriedade e um valor separados por dois pontos (:) e terminados por um ponto e vírgula (;).

Sintaxe Básica:

Há três tipos principais de seletores:

  • Por Elemento: Selecionam todos os elementos daquele tipo. É declarado com o mesmo nome da tag HTML;

  • Por Id: Selecionam apenas o elemento com o id indicado. É declarado começando com #;

  • Por Classe: Selecionam todos os elementos com a classe especificada. É declarado começando com ponto.

Há três formas de inserção do CSS:

  • Inline: O CSS é inserido diretamente nos elementos HTML usando o atributo style.

  • Embutido: O CSS é incluído dentro da tag style no cabeçalho head do documento HTML.

  • Externo: : O CSS é colocado em um arquivo separado com a extensão .css e, em seguida, incluído no documento HTML usando a tag < link>.

  1. O que é Box Model?

O Modelo de Caixa (ou Box Model) é um conceito fundamental no CSS que nos descreve, visualmente, como um elemento HTML é representado em tela.

Cada elemento é ilustrado como uma caixa retangular que pode ter margens, bordas, preenchimento e o conteúdo interno:

Onde:

  • Conteúdo (content): é o conteúdo real dentro da caixa. Pode ser um texto, uma imagem, etc;
  • Preenchimento (padding): é usado para criar um espaçamento entre conteúdo e a borda;
  • Borda (border): é usada para criar uma separação visual ao redor do elemento;
  • Margem (margin): é usada para criar espaçamento entre a borda e os elementos vizinhos.

  1. O que é FlexBox?

O Flexbox é um recurso do CSS que vai nos ajudar a criar layouts mais flexíveis e responsivos;

Com ele é possível organizar e alinhar os elementos de uma página de uma maneira dinâmica.

Container Flexível:

  • Um elemento HTML que contém itens flexíveis;
  • É transformado em um container flexível usando a propriedade display: flex :

Itens Flexíveis:

  • Os elementos filhos diretos do container flexível são chamados de itens flexíveis;
  • Eles se comportam de acordo com as propriedades do Flexbox.

Quando usamos o FlexBox, liberamos algumas propriedades específicas que só podem ser usadas em containers flexíveis:

  • Justify-Content: é responsável por definir como os elementos devem ser alinhados ao longo do eixo principal (de maneira vertical);
  • Align-Items: usamos para alinhar os itens no eixo transversal (de maneira horizontal).
